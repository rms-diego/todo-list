## 🚀 Instalando Todo List

Para instalar o Todo List, siga estas etapas:

</br>

**Linux, macOS e Windows :**

**Clone o repositório**

```
  git clone https://github.com/rms-diego/todo-list.git && cd todo-list
```

Depois execute os seguintes comando

**Npm**

```
  npm i && npm run dev
```
